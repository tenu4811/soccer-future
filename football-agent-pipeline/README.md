# 足球分析多智能体流水线

> 将长链足球分析分解为 19 个独立阶段，分发给多个 AI 智能体并行/串行执行，彻底解决单会话上下文衰减。

## 解决的问题

大型语言模型在进行长文档分析时，上下文窗口越大，输出质量越差（中段迷失、首因/近因效应、逻辑断裂）。足球深度分析报告动辄 5k-10k 字，传统单会话方式无法稳定产出高质量内容。

**方案**：将分析流程拆解为 19 个独立阶段（每个阶段一个独立 Agent 提示词），通过 `manager.md` 编排执行顺序，配合 3 道机器门禁确保输出质量。

## 架构

| 工具 | 职责 |
|------|------|
| **Claude Code** | 编排引擎 + 门禁检查 + 深度分析（泊松/贝叶斯/思维链/对位） |
| **Hermes** | 浏览器数据采集（FBref/Transfermarkt/WhoScored 直访） |
| **OpenCode** | 文档工程 + 数据验证 + 质量评分 |

### 执行流程概览

```
数据采集 (阶段0-1b) → 门禁1 → 分析链 (阶段1.5-1.8) → 对位分析 (阶段2-2.5)
→ 门禁2 → 宏观分析 (阶段2a-2g) → 报告写作 (阶段3) → 门禁3 → 验证 (阶段4)
```

## 前置依赖

- [Claude Code](https://docs.anthropic.com/en/docs/claude-code/overview)
- [Hermes](https://hermes.com/) 或任何支持浏览器自动化的 CLI 工具
- Python 3.8+（用于泊松预测脚本）
- 数据源：FBref、Transfermarkt、WhoScored（公开可访问）


## 质量保障系统

### 3 道机器门禁

| 门禁 | 时机 | 检查项 | 工具 |
|------|------|--------|------|
| **门禁1** | 数据采集完成后 | D1(估算标记)、D5(宏观变量)、D9(数据源多样性) | Bash grep |
| **门禁2** | 对位分析完成后 | D2(博弈点指向性)、D3(反指标完整性)、D7(球员速查卡) | Bash grep |
| **门禁3** | 报告写作完成后 | D8(字数)、D10(英文清零)、总分≥88 | check-all.sh |

### 11 个评分维度（D1-D11）

| 维度 | 内容 | 检查方式 |
|------|------|----------|
| D1 | 估算标记（"估/~/约/大约"） | `scripts/check-d1-estimates.sh` |
| D2 | 博弈点指向性 | 门禁人工复核 |
| D3 | 反指标完整性 | 门禁人工复核 |
| D4 | 模板套话/陈词滥调 | `scripts/check-d4-cliches.sh` |
| D5 | 宏观变量覆盖（≥5种） | 门禁自动检查 |
| D6 | 数据源引用 | 人工巡检 |
| D7 | 球员速查卡完成度 | 门禁自动检查 |
| D8 | 字数（≥5000） | `scripts/check-d8-charcount.sh` |
| D9 | 数据源多样性（≥5种标签） | `scripts/check-d9-tags.sh` |
| D10 | 英文清零（无拉丁姓名残留） | `scripts/check-d10-latin.sh` |
| D11 | YAML 格式完整性 | `scripts/check-d11-yaml.sh` |

### 防复发硬规则（R1-R5+R8）

基于实战审计教训（2026-07 葡克审计报告：75.6 分，D10 门禁失败）制定的 6 条强制执行规则，详见 `rules/` 目录。

## 项目结构

```
├── manager.md        # 管理器运行指令（19 阶段执行序列）
├── config.yaml       # 比赛参数配置
├── agents/           # Agent 提示词（20 个独立文件，每阶段一个）
├── scripts/          # 机器自检与计算脚本（shell + Python）
├── rules/            # 防复发硬规则
├── outputs/          # 报告产物（含完整实战示例）
└── reference/        # 参考文件（预留）
```

## 示例输出

`outputs/2026-07-10-法国vs摩洛哥/` 包含一场完整比赛的 19 份阶段分析报告，最终报告评分 **91.3/100**，约 87,000 字。
