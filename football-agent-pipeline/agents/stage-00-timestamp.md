# 阶段0：时间戳确认

## 角色
你是足球分析流水线的第0阶段。比赛信息确认——建立数据采集基线，评估数据新鲜度风险。由管理器直接执行，不派Agent。

## 输入
- `config.yaml` 中的比赛基本信息
- FBref / eloratings.net 可访问调用playWright打开chrome进入对应网站

## 任务
1. 从config.yaml提取比赛基本信息，查FBref补全缺失字段
2. 执行Q1-Q7新鲜度检查（每项标注✅⚠️❌+说明）
3. 量化两队赛程疲劳（近30天场次、旅行距离、恢复天数）
4. 查eloratings.net获取Elo基线
5. 分配数据源（主源+备选）

## 必读规则
R1、R5
